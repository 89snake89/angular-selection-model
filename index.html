Angular Selection Model

Select Something

Without the extra junk

Basics

Getting up and running is a snap.

Include the selectionModel script on your page and add it as a depedency to your angular app.

The selectionModel directive works with ngRepeat here's a basic example:

Controller

Choose something fancy!

  • {{$index+1}} {{item.label}}

Someone so fancy would choose the {{fancy.selectedItems[0].label | lowercase}}.

Controller...
myApp.controller('FancyCtrl', function() {
  this.bag = [
    {label: 'Monacle', selected: false},
    {label: 'Top Hat', selected: true},
    {label: 'Mustachio', selected: false},
    {label: 'Coffee', selected: false}
  ];
});
View...
<ul ng-controller="FancyCtrl as fancy">
  <li ng-repeat="item in fancy.bag" selection-model>
    {{$index+1}} {{item.label}}
  </li>
</ul>

More...

There are tons of ways to configure the selectionModel directive. You can:

  • Work with multi-selections and checkboxes
  • Programmatically get at and update the selection
  • Use all the usual ngRepeat partners in crime - filters, paging, ... already built into Angular.

The selectionModel directive is built to manage your selections and get out of the way. Use your own list and table styles and don't jump through hurdles to skin complex widgets.

For additinal examples and documentation check out the GitHub repo.
